## [1.0.0] - 2025-06-14

### Added | مضاف
- Initial release of Iraq WDI Analysis
- Jupyter notebook with complete analysis pipeline
- Data cleaning, feature engineering, and baseline modeling
- Exploratory data analysis for Iraq's development indicators
- Historical era classification
- Trend analysis for key indicators
- Professional bilingual report

### Features | الميزات

- 66 years of data (1960-2025)

- Linear trend models
- Nearest-centroid era classifier
- Naive forecasting evaluation
